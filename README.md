Aneeq Zia, Kiran D. Jarc. ’Surgical Visual Domain Adaptation: Results from the MICCAI 2020 SurgVisDom Challenge’. ArXiv:2102.13644, 2021. https://doi.org/10.48550/arXiv.2102.13644.
